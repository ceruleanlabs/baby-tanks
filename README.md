# BABY TANKS

## Setup

    $ npm install
    $ npm install -g grunt grunt-cli

## Development

    $ grunt serve

## Deploying to GitHub Pages

First make sure the game is built. Normally it will be, but if it's not:

    $ grunt build
    $ grunt gh-pages
