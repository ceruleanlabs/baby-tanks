# BABY TANKS

## Setup

    $ npm install
    $ npm install -g grunt grunt-cli

## Development

    $ grunt serve

## Deploying to GitHub Pages

First make sure you have gh-pages locally.

    $ git checkout gh-pages
    $ git checkout master

Then deploy.

    $ grunt gh-pages
